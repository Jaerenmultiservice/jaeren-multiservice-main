# Videoer for tjenestekort

Legg inn disse filene her (MP4 eller WebM, loopet, 10–30 sek):

- **takvask.mp4** – Person som vasker tak (drone/høyden)
- **hoytrykk.mp4** – Person som spyler terrasse/oppkjørsel
- **hekkeklipp.mp4** – Person som klipper hekk med hekkeklipper
- **gravearbeid.mp4** – Gravemaskin i arbeid

Krav: autoplay, muted, loop. Ekte håndverker-arbeid, mørk/industriell stemning.
