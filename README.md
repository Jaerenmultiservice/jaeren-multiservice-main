# Jæren Multiservice

Nettside for Jæren Multiservice.

- **Produksjon:** [https://www.jaerenmultiservice.no/](https://www.jaerenmultiservice.no/)
- **Repo:** kobles til GitHub; deploy skjer via din hosting (f.eks. Netlify) når du pusher til `main`.

```bash
npm install
npm run dev
npm run build
```
